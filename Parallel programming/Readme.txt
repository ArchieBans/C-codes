
The program final_fit.c calculate the best L1 norm fit as requested in the assignment specifications. 
NOTE: If you think it is taking too long please for 10 years run just let it run, it will finish within 15 minutes. And it will give a correct solution. 

Usage of the program:

Please provide extra parameter between 0-5 to get the values.
For example :  ./dataPar 2

Details for 0-5:
0 : First six points.
1 : First ten points.
2 : First fourteen points.
3 : First eighteen points.
4 : Year 2002.
5 : All 10 years.
